ClimatePros Stream 2 — Architecture Blocks Reference

TECHJAYS  ·  CLIMATEPROS ENGAGEMENT  ·  STREAM 2 POC  ·  Quick reference — what each block does

Clients
BlockWhat it does
FieldJetX Web App The screen the estimator uses to upload drawings and review results
Manager / Admin Manager adds feedback on bids · Admin manages users and rate cards
Existing · FieldJetX Platform  (Azure · unchanged · reused)
BlockWhat it does
Front Door + WAF The main gate — blocks bad traffic, handles secure connection (TLS)
App Gateway The traffic director — routes requests to the right service
FJX Auth · Session Tokens The security guard — checks who you are and what you're allowed to do on every single request
Blob Storage The file cabinet — stores all uploaded drawings, generated quotes and images
VNet + Private Endpoints The private tunnel — keeps all data off the public internet
Key Vault The safe — stores all passwords, API keys and secrets; nothing is hardcoded in code
New Infrastructure · This Project  (Azure · Container Apps)
BlockWhat it does
FastAPI + WebSocket API The brain — receives all requests, coordinates processing, sends back results via WebSocket or HTTP polling
Azure Service Bus The queue — holds analyze jobs and delivers them to workers reliably; retries on failure
KEDA Workers · Celery 5.6 The workforce — scales up automatically, processes all pages in parallel, calls AI models, merges results, runs pricing
PostgreSQL 16 The main database — stores bids, extracted items, pricing, audit trail and every AI call log
Redis Cache The short-term memory — tracks live job progress and caches repeated results to avoid re-running models
Knowledge Graph DB The learning system — remembers past corrections to make future extractions smarter over time
Cloud AI Verifiers  (suggested for accuracy · models subject to change)
BlockWhat it does
Gemini 3.5 Flash ⚗ Experimental Google's AI — double-checks Zeus's extraction independently via cloud API (Google Vertex AI)
Claude Fable 5 ⚗ Experimental Anthropic's AI — does the same independent check simultaneously via cloud API; higher token usage, can be swapped based on cost
Private Model Infra · ClimatePros Client  (on-prem · Mac Studio "Tundra")
BlockWhat it does
Zeus — Primary Extractor The main AI on the client's own machine — reads drawings and pulls out quantities, equipment and statuses. Runs Claude Sonnet 4.6, grounded in prior FieldJetX bid history
Observability & Tracing
BlockWhat it does
Monitor + App Insights The CCTV and alarm system — records logs, traces and metrics from every block; fires alerts when something goes wrong or costs spike
